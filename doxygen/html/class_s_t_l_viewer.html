Detailed Description

The STLViewer class provides a main application window.

STLViewer has its own layout to which added QToolBars, QMenuBar, QStatusBar, AxisGroupBox, DimensionsGroupBox,MeshInformationGroupBox and and PropertiesGroupBox objects STLViewer have multiple (MDI) document interface
